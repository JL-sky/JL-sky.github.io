github提交

  • github创建项目
  • 复制SSH链接
  • git clone “链接”（克隆项目到本地）
  • 将项目文件复制到克隆的文件夹中
  • 使用git进入项目文件夹
  • git add . (将项目文件添加到仓库)
  • git commit -m “提交提示信息”（提交项目文件到仓库）
  • git push -u origin main

注：git可以一次提交多个项目文件，因此可以在本地添加多个项目文件后一次性提交

github使用技巧

  • 在线查看项目
    进入项目详情页按下”.”键，GitHub会自行使用在线vscode查看项目文件代码而不需要下载到本地
  • 在线运行项目
    在项目详情页地址栏”https://“后加入”gitpod.io/“即可进入在线vscode运行查看的项目
